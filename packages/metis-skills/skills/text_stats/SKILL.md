# text_stats

Count the words, characters, and lines in a block of text.

- **Category:** data_cleanup
- **Permissions:** none (no network, connectors, secrets, or filesystem access)
- **Approval:** not required (read-only analysis)

## Input

```json
{ "text": "the text to analyze" }
```

## Output

```json
{ "words": 3, "characters": 18, "lines": 1 }
```
